- How does emulatorJS work?

- How does Webretro (AKA Retroarch, AKA Rarch) work?

Go to Any ROM website (ex. https://romsgames.net/) or go into games page (for some roms I've uploaded) and download the game you wanna play. Then, after it finishes the download, DO NOT OPEN THE FILE, go instead to the games page, click on Webretro Button (the one with a white arcade alien in a black background), then, when the page loads, click on "AutoDetect (Slower to Load)" (IMPORTANT NOTE: Do this step so the ROM file is correctly executed in the corresponding core (AKA platform) so there are no errors), then click on the "Upload File" button, search for your ROM (generaly in ".zip" format), select it and click on "Open". Tadaaa! It will start loading!
